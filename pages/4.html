Text

It's about marking up the text, right?

According to Wikipedia, Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. I guess we should start by looking at some text tags.

Content on a page can be divided into a hierarchy of headings. HTML has 6 heading tags that can be used.

< h1 >

< h2 >

< h3 >

< h4 >

< h5 >
< h6 >

Browsers will all render the size if a heading tag slightly differently, but < h1 > will always be largest, and < h6 > will always be smallest.

What about the rest of your content? Use < p > on all your text that isn't a heading. Don't use heading tags for your body text, leave them for headings. When in doubt, use < p > on your body text. Because of the way HTML is interpreted by browsers, font size was controllable in HTML. The preferred way of controlling font face and size is with CSS ... more about that later.
