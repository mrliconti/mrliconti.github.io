This lesson is about the basic structure of HTML.

The page begins with the < html > tag. The < head > section of the page allows you to contain further tags, like < title > for instance.

When the title section is done, the < title > closes and then we can close the < head > section so that the body tag < b > can hold our information. We end the document by closing first the < b > then the < html >

< html >
