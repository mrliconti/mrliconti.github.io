Introduction to HTML

You can learn about HTML any way you want. I've never seen a book as good as Jon Duckett's HTML & CSS Design and Build Websites. If you're looking for a book, this is it. My digital notes/scribbles are all based on Duckett's.

If you want to see where it all comes from, w3.org's HTML & CSS standards would be the place.

I also like the Mozilla Foundation's MDN Web Docs.
